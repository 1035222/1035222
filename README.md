## 🚀 Proyecto destacado: NOUSIA

Plataforma académica modular full-stack (React + Node.js/Express + MySQL), organizada por roles institucionales:

| Módulo | Rol | Función |
|--------|-----|---------|
| 🟢 **SONAR** | Estudiante | Notas, tareas, horarios y certificados |
| 🔵 **SAICO** | Docente | Grupos, asignaciones, entregas y calificaciones |
| 🟣 **ZAFIRO** | Administración / Soporte | Usuarios, roles, tickets, reportes y horarios |

Backend construido con Express 5, cors, dotenv, mysql2 y nodemon para el entorno de desarrollo. Autenticación JWT firmada manualmente con HMAC SHA-256, hash de contraseñas con `crypto.scrypt`, validación con Zod, rate limiting personalizado, e integración con la API de OpenAI para un asistente académico con fallback local.
